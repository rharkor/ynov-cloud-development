## Getting Started

1. Install dependencies

```bash
npm install
```

2. Run the server

```bash
npm start
```

3. Open the browser and navigate to `http://localhost:3000`

## Stack

- Dev container
- Unit tests
- Integration tests
- E2E tests
- Linting
- Code formatting (Prettier)
- Next.js app router
- Tailwind CSS
- TypeScript
- Nextui (UI components)
- Trpc (RPC)
- Prisma (ORM)
- MongoDB (Database)
